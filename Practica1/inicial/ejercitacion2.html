  2. 2.b.

    Este segmento es un bloque DIV, se puede insertar en el body/footer. En este caso tiene un atributo ID con valor bloque1. Este atributo NO es obligatorio y sirve para luego identificar este bloque en Javascript o CSS

  3. 2.c.

    Este segmento es un bloque IMG, se puede insertar en el body/footer y se utiliza para mostrar imagenes. En este caso incluye los atributos:

    • src, con valor vacio
    • alt, con valor "lugar imagen"
    • id, con valor im1
    • name, con valor im1
    • width, con valor 32
    • height, con valor 32
    • longdesc, con valor detalles.htm
    Solo son obligatorios los atributos src y alt

  4. 2.d.

    El atributo meta define metadata (informacion sobe los datos), siempre va en el Head del documento, en este caso se define un nombre para estos metadatos (name) "keywords", un lenguaje (lang) "es", el valor asociado a otros contenidos o nombres (content) "casa, compra, venta, alquiler"
    El segundo tag meta especifica un header para la informacion del atributo content (http-equiv) "expires" (content)"16-Sep-2019 7:49 PM""
    Ningun atributo es obligatorio

  5. 2.e.

    El atributo a define enlaces a otras paginas, el atributo href especifica la url a la que dirige el link "http://www.e-style.com.ar/resumen.html", el atributo type especifica el formato del documento linkeado, en este caso "text/html", hreflang especifica el lenguage del documento linkeado "es", charset especifica el formato de los caracteres usados "utf-8", rel especifica la relacion entre el documento actual y el linkeado "help"
    Ningun atributo es obligatorio, en el caso de no tener href el atributo a es solamente un placeholder de un enlace.

  6. 2.f.

    El atributo table define una tabla en la cual se puede formatear datos en las filas y columnas, la tabla tiene un ancho de 200 pixeles (width), el atributo summary esta obsoleto y no deberia usarse, explicaba el formato de la tabla.
    El tag caption define el Titulo de la tabla que el usuario puede ver, "Titulo", con su atribo align lo alinea en un lugar de la tabla, "top" arriba en este caso.
    El tag tr define una fila de la tabla y el tag td una celda de la tabla. El tag th define una celda de encabezado. El atributo scope especifica que las celdas de encabezado son encabezados de columnas,filas o grupos de estas, en este caso de columnas "col" y filas "row".
    Ninguno de estos atributos son obligatorios.
