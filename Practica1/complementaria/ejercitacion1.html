Ejercicio 1 complementaria

  1. ¿Ventajas de HTML5?

    • Es gratuito

      No se requiere ningun programa especial (costoso) para empezar a programar en HTML5, hasta se puede programar en un bloc de notas.

    • Código más ordenado

      Gracias a las nuevas etiquetas que permiten nombrar partes de la estructura básica de una página web, podemos separar el código entre etiquetas y contenido, lo que nos permite trabajar de manera más efectiva.

    • Compatibilidad en navegadores

      Sin importar que navegador empleen los usuarios, el contenido se podrá visualizar correctamente

    • Almacenamiento mejorado

      HTML5 incorporó el almacenamiento local, el cual se define a sí mismo como "mejor qu las cookies", ya que la información nunca se transfiere al servidor. De esta manera, la información se mantiene segura. Esto nos permite mantener la información aunque el usuario cierre el navegador o borre las cookies, y hasta nos permite mejorar el tiempo de respuesta de la aplicación gracias al uso de caché

    • Diseño adaptativo

      HTML5 es compatible con los navegadores móviles, por lo que cada página que se crea para computadoras en HTML5, se puede adaptar a los dispositivos móviles

  2. ¿Por qué utilizarlo?

    El lenguaje HTML5 se usa para definir la estructura básica de una página web. Sin embargo, una de sus más grandes adiciones en esta nueva versión es poder añadir audio y video sin necesidad de usar Flash u otro reproductor multimedia.
    Por medio de las etiquetas < video>y < audio> de HTML5, permite añadir videos o audio sin necesidad de usar Adobe Flash o cualquier otro plugin de tercero. Toda la acción sucede desde el propio navegador, lo que puede ayudar a disminuir al tamaño del archivo final de tu página.
    La API para esta etiqueta permite dibujar elementos en 2D y animarlos. El resultado final bien podría incluirlo en la página de inicio del sitio web pues la API da bastante control sobre los elementos.
    También la API permite añadir eventos de teclado, ratón y cualquier otro mando que desee incluir. Esta posibilidad ha emocionado a muchos desarrolladores que se han dedicado a realizar sus juegos en HTML5. Pudiendo jugar desde cualquier navegador.
    Por lo tanto, HTML5 nos permite una mayor interacción entre nuestras páginas web y contenido multimedia, ademas de una mayor facilidad a la hora de codificar nuestro diseño básico
